Notes

        
    
          
  • The server only handles GET — all other methods return 405 Method Not Allowed.
    • 
          
  • A malformed HTTP request returns 400 Bad Request (testable via curl or netcat).
    • 
          
  • Every response includes Content-Type (auto-detected from file extension) and Content-Length.
    • 
          
  • Request bodies are parsed using Content-Length and classified as text or binary via Content-Type.
    • 
          
  • The read buffer grows dynamically — headers up to 8 KB, body up to 1 MB.
    • 
          
  • Bind address and port are configurable: HOST=0.0.0.0 PORT=8080 cargo run.
